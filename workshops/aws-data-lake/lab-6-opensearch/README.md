# Lab 6: Search and Dashboards with OpenSearch

Ingest data into Amazon OpenSearch Service, run full-text and aggregation queries, and build a visualization dashboard.

## Lab guides

| File | Purpose |
|---|---|
| [student-lab-6-opensearch.md](student-lab-6-opensearch.md) | Main student lab: create an OpenSearch domain, ingest documents, run queries |
| [opensearch-rbac-lab.md](opensearch-rbac-lab.md) | RBAC lab: index-level access control with roles, DLS, and field-level security |
| [opensearch-lineage-lab.md](opensearch-lineage-lab.md) | Lineage lab: capture and visualize data pipeline lineage using nodes, edges, and run history |
| [opensearch-data-setup.md](opensearch-data-setup.md) | Data loading guide: index sample documents into your domain |
| [opensearch-visualization-setup.md](opensearch-visualization-setup.md) | Dashboard guide: build visualizations in OpenSearch Dashboards |

## What you'll learn

- How to create and configure an OpenSearch domain
- How to index documents and run search queries using the REST API
- How to build bar charts, pie charts, and time-series visualizations in OpenSearch Dashboards
- How to create roles with index-level, document-level, and field-level access control (RBAC lab)
- How to model and query data lineage (provenance, impact analysis, freshness, orphan detection) using OpenSearch indices

## Prerequisites

- AWS Console access and your `<USER>` slug from your instructor
- This lab is standalone: Lab 1 is not required
