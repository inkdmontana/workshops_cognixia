# Chapter 1: Vibe Coding Fundamentals

Learn how to use AI assistants effectively to build real software: not just generate snippets, but drive full application development through structured prompting.

## Contents

| File | Description |
|------|-------------|
| [prompts.md](./prompts.md) | Starter prompts for building a FastAPI app with AI |

## Learning Outcomes

- Understand what "vibe coding" means in a professional context
- Write effective prompts that produce working, structured code
- Build a basic FastAPI app from scratch using AI assistance
- Know when to guide the AI vs. when to trust its output
